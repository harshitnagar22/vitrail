# Vitrail
Kuartz's Design system view components.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "vitrail"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install vitrail
```

## Development

watch tailwindcss changes with:
```bash
tailwindcss_watch
```
